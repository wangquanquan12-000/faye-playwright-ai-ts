# test-media

发帖用例从此目录随机选取 **1～9 张图片**，通过发帖页 **Image** 按钮上传。

## 支持格式

`.jpg`、`.jpeg`、`.png`、`.webp`、`.gif`

## 推荐文件名（请手动放入本目录）

| 文件名 | 说明 |
|--------|------|
| `post-01.jpg` | 测试图片 1 |
| `post-02.jpg` | 测试图片 2 |
| `post-03.jpg` | 测试图片 3 |
| `post-04.jpg` | 测试图片 4 |
| `post-05.jpg` | 测试图片 5 |
| `post-06.jpg` | 测试图片 6 |
| `post-07.jpg` | 测试图片 7 |
| `post-08.jpg` | 测试图片 8 |
| `post-09.jpg` | 测试图片 9 |

文件名不必严格对应上表，但建议至少准备 9 张图片以便随机组合。视频文件（`.mp4` 等）不会被自动选用。
